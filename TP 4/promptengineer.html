Prompt Engineer

Description : Le Prompt Engineer conçoit des requêtes optimisées pour interagir efficacement avec les intelligences artificielles génératives.

Missions :

  • Rédaction de prompts précis pour l'IA
  • Test et ajustement des réponses générées
  • Collaboration avec les équipes techniques pour améliorer les modèles

Salaires :

Après un BTS SIO : Environ 50 000 à 55 000 € brut/an

Après une école d'ingénieur : Environ 60 000 à 65 000 € brut/an
